# Web Chat Server

Welcome to the **Web Chat Server**! This project implements a simple and efficient web-based chat server that allows multiple users to send and receive messages in real time. It uses basic web technologies like HTTP and WebSockets to provide a lightweight solution for real-time communication. Whether you're building a small web application or just experimenting with web servers, this repository is the perfect place to start.
